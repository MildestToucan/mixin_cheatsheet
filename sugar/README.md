# What Is Sugar?

<- [Back to Main](/README.md)

As described in the [Sugar](https://github.com/LlamaLad7/MixinExtras/wiki/Sugar) page of the MixinExtras Wiki,
Sugar in the context of MixinExtras describes extra parameters that can be appended to any injector handler to provide
additional functionality that would otherwise be tedious to implement, or require less functional and/or convenient tooling.

In that sense it extends a bit beyond purely syntactic sugar, as it gives more optimized approaches to certain things in Mixin.
